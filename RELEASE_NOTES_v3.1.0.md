How to create the GitHub Release locally (recommended)
-----------------------------------------------------
Using GitHub CLI (`gh`):

```
gh release create v3.1.0 --title "v3.1.0" --notes-file RELEASE_NOTES_v3.1.0.md
```

Or with `curl` and a Personal Access Token (set `GITHUB_TOKEN` in your environment):

```powershell
$env:GITHUB_TOKEN="YOUR_TOKEN"
$body = @{
  tag_name = "v3.1.0"
  name = "v3.1.0"
  body = Get-Content -Raw RELEASE_NOTES_v3.1.0.md
  draft = $false
  prerelease = $false
} | ConvertTo-Json
Invoke-RestMethod -Uri "https://api.github.com/repos/HDGamer-png/Chatbot-langgraph/releases" -Method Post -Headers @{ Authorization = "token $env:GITHUB_TOKEN"; Accept = "application/vnd.github+json" } -Body $body -ContentType "application/json"
```
